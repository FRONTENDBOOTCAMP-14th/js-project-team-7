> 심플하고 직관적인 UI에서 여행지를 검색하고, 추천을 받아보세요!

---

## 프로젝트 소개  
**7대륙**은 사용자가 원하는 여행지를 빠르게 탐색하고, 맞춤형 추천을 받을 수 있는 간단한 웹 서비스입니다.  
외부 라이브러리 없이 **HTML + CSS + JavaScript(ES6+)**만으로 구현했습니다.

## 🛠 기술 스택  
- **Frontend** : HTML5, CSS3, JavaScript (ES6+)

- **Storage** : LocalStorage
